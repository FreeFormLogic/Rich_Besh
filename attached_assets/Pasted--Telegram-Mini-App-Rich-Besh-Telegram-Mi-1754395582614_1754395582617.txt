# Задание: Доработка Telegram Mini App для блогера Rich Besh

## Контекст проекта
Это Telegram Mini App для популярного блогера Rich Besh, который занимается:
- Продажей платных спортивных прогнозов
- Партнёрскими программами букмекеров
- Доверительным управлением/раскруткой счёта
- Обучающими курсами и мотивационными гайдами
- Рекламой брендов и инфлюенс-маркетингом
- Платными консультациями

## Целевая аудитория
Люди, которые мечтают о роскошной жизни, хотят быстро заработать без традиционной работы. Они вдохновляются успехом и готовы инвестировать в обучение и прогнозы.

## Основные задачи

### 1. Переработка раздела Stories
- Удалить текущие заглушки и неработающие видео
- Интегрировать реальный контент из Instagram блогера
- Использовать видео из предоставленного списка CDN
- Создавать автоматические обложки из кадров видео с учетом ориентации (вертикальная/горизонтальная)

### 2. Переработка текстового контента
- Загрузить и использовать тексты из https://richbesh.b-cdn.net/IG/posts_info.csv
- Убрать все упоминания "показухи", "видео-доказательства" и подобные негативные формулировки
- Заменить на вдохновляющие тексты о:
  - "Путь к успеху"
  - "Результаты учеников"
  - "Lifestyle достижения"
  - "Инвестиции в себя"
  - "Финансовая свобода"

### 3. Интеграция медиа-контента
Использовать видео и фото из CDN для создания:
- Галереи успеха (дорогие автомобили, путешествия, рестораны)
- Мотивационных историй
- Примеров результатов от прогнозов
- Визуальных доказательств экспертности

### 4. Улучшение UX/UI
- Сделать плавные переходы между разделами
- Добавить эффекты появления для премиум-контента
- Оптимизировать загрузку видео (lazy loading)
- Добавить индикаторы загрузки
- Сделать адаптивный дизайн для всех размеров экранов

### 5. Структура контента
Организовать контент по категориям:
- **Мой путь к успеху** - личная история Rich Besh
- **Результаты учеников** - истории успеха подписчиков
- **Эксклюзивные прогнозы** - раздел с платным контентом
- **Обучение** - курсы и мастер-классы
- **Lifestyle** - мотивационный контент о роскошной жизни

### 6. Технические требования
- Оптимизировать производительность приложения
- Настроить корректную работу с Telegram WebApp API
- Добавить аналитику действий пользователей
- Реализовать систему уведомлений о новых прогнозах
- Добавить интеграцию с платежными системами

### 7. Тональность и стиль
- Премиальный, но доступный
- Мотивирующий, без навязчивости
- Фокус на возможностях, а не на проблемах
- Использовать слова: успех, достижения, результаты, возможности, инвестиции в себя

### 8. Конкретные правки текстов
Заменить:
- "Доказательства Rich Life" → "Lifestyle успешных людей"
- "Видео-доказательства" → "Истории успеха"
- Любые упоминания "показухи" → "достижения", "результаты"

### 9. Работа с видео
- Для каждого видео создать превью из наиболее ярких кадров
- Добавить описания к видео на основе данных из CSV
- Группировать видео по темам: автомобили, путешествия, рестораны, события
- Реализовать видеоплеер с контролами и полноэкранным режимом

### 10. Call-to-Action элементы
Добавить явные призывы к действию:
- "Получить VIP-прогноз"
- "Начать зарабатывать"
- "Присоединиться к команде победителей"
- "Инвестировать в свое будущее"

## Файлы для работы
Используй контент из CDN по предоставленным ссылкам. Все видео и изображения доступны для интеграции. CSV файл содержит описания и метаданные для каждого поста.

## Результат
Премиальное приложение, которое вдохновляет пользователей на достижение финансового успеха через спортивные прогнозы и обучение у Rich Besh. Каждый элемент должен подчеркивать экспертность и успешность блогера.
